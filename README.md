# SoundCloud Artwork Copier

Tampermonkeyユーザースクリプト。SoundCloudのトラックページのヘッダー、検索ボックスの右にボタンを追加し、クリックするとそのトラックのジャケット画像をクリップボードにコピーする。また、お気に入り一覧などのトラック一覧ページでは、各ジャケットの右下にホバー時のみ表示されるコピーボタンを設置する。

## インストール

1. Tampermonkeyをインストールする。
2. Tampermonkeyのダッシュボードで「新規スクリプトを作成」を選び、`soundcloud-artwork-copier.user.js`の内容を貼り付けて保存する。
3. `soundcloud.com`のトラックページを開くと、ヘッダーの検索ボックスの右にボタンが表示される。
4. `https://soundcloud.com/you/likes`などのトラック一覧ページでは、各ジャケットにマウスを乗せると右下にコピーボタンが表示される。

## 動作条件

- ログイン済みのSoundCloudヘッダーを対象とする。
- ヘッダーのボタンはトラック単体ページ（`og:type` が `music.song`）でのみ機能する。プレイリストやプロフィールページではボタンを押すと失敗表示になる。
- 一覧ページのジャケットオーバーレイボタンは、サムネイル画像がまだ遅延読み込みされていない場合は失敗表示になる。
