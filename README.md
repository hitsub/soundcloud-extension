# SoundCloud Artwork Copier

Tampermonkeyユーザースクリプト。SoundCloudのトラックページのヘッダー、検索ボックスの右にボタンを追加し、クリックするとそのトラックのジャケット画像をクリップボードにコピーする。また、お気に入り一覧などのトラック一覧ページでは、各トラックの既存のホバーアクションメニュー（Like/Follow/Moreなど）に並べてコピーボタンを設置する。グリッド表示（Badges）とリスト表示（List）の両方に対応する。

## インストール

1. Tampermonkeyをインストールする。
2. Tampermonkeyのダッシュボードで「新規スクリプトを作成」を選び、`soundcloud-artwork-copier.user.js`の内容を貼り付けて保存する。
3. `soundcloud.com`のトラックページを開くと、ヘッダーの検索ボックスの右にボタンが表示される。
4. `https://soundcloud.com/you/likes`などのトラック一覧ページでは、各トラックのLike/Follow/Moreなどのボタン列にコピーボタンが並ぶ。
5. トラックの「More」メニューを開くと、既存の「Download file」の下に「Download file with metadata」が追加される。クリックすると、ファイルをダウンロードする際にタイトル・アルバム名（トラックのタイトルと同一）・アーティスト名（WAVはさらにジャケット画像も、MP3はID3v2タグとして。未設定の項目だけを埋める）を書き込んでから保存する。

## 動作条件

- ログイン済みのSoundCloudヘッダーを対象とする。
- ヘッダーのボタンはトラック単体ページ（`og:type` が `music.song`）でのみ機能する。プレイリストやプロフィールページではボタンを押すと失敗表示になる。
- 一覧ページのジャケットオーバーレイボタンは、サムネイル画像がまだ遅延読み込みされていない場合は失敗表示になる。
- 「Download file with metadata」はWAVとMP3にメタデータを書き込む（WAVはタイトル/アーティスト/アルバムのみ、MP3はジャケット画像も含む）。それ以外の形式（m4a/flacなど）は無加工でダウンロードされる。
- MP3のタグ書き込みには外部ライブラリ`browser-id3-writer`（バージョン固定・ソース確認済み）をCDNから動的読み込みする。
